# 📊 Applied Statistics & Data Analysis Project

## 🎯 Project Overview

This project aims to apply fundamental statistical concepts to real-world datasets in order to extract meaningful insights and support data-driven decision-making.

The project is divided into two main analytical parts:

* Health Data Analysis
* Automotive (MPG) Data Analysis

---

## 🧠 Phase 1 – Statistical Foundations

This phase focuses on understanding key statistical concepts:

* Descriptive statistics (mean, median, variance, standard deviation)
* Types of variables (discrete vs continuous)
* Probability distributions
* Central Limit Theorem (CLT)
* Sampling and estimation
* Hypothesis testing (H0, H1, p-value)

---

## 📊 Phase 2 – Health Dataset Analysis

### 🔍 Objectives

* Analyze health spending and life expectancy
* Identify relationships between variables
* Apply statistical tests

### ⚙️ Analysis Performed

* Variable classification
* Descriptive statistics
* Data visualization (histograms, boxplots, line plots)
* Correlation analysis (Pearson)
* Hypothesis testing

### 📈 Key Findings

* Positive correlation between health spending and life expectancy
* Life expectancy improves over time
* Significant differences between countries
* Spending increases do not always lead to proportional gains in life expectancy

---

## 🚗 Phase 3 – MPG Dataset Analysis

### 🔍 Objectives

* Analyze vehicle characteristics
* Identify trends over time
* Understand relationships between performance and efficiency

### ⚙️ Analysis Performed

* Analysis of mpg, cylinders, origin
* Relationship between horsepower and mpg
* Time-based analysis using model_year
* Group comparisons (origin, cylinders)

### 📈 Key Findings

* Increase in mpg → improved fuel efficiency over time
* Decrease in weight and number of cylinders
* Negative relationship between horsepower and mpg
* Japanese cars are more fuel-efficient
* American cars are heavier and more powerful

---

## 📌 Global Conclusion

The analysis reveals a general trend toward efficiency and optimization:

* In health: improvement in life expectancy
* In automotive: lighter, more efficient, and less fuel-consuming vehicles
